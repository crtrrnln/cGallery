# 01 - Vision

## Core Identity
cGallery is a **fast, local-first Android gallery application** designed for advanced physical organization of large media libraries. It prioritizes direct control over the file system, high-performance visual experiences, and a streamlined triage workflow for new content.

## The Problem
Standard Android galleries are often logical-only, hiding the physical reality of the file system. Users who want to maintain a clean directory structure are forced to use file managers, which lack a rich media experience. This leads to "Inbox Fatigue"—the accumulation of thousands of photos in various download or camera folders that are never properly filed.

## The Solution: Physical-First Curation
cGallery replaces manual file management with an integrated **Inbox-to-Archive** workflow:

1.  **Detection (Inbox System):** Automatically identifies new media in monitored directories and queues them for triage.
2.  **Curation (Gallery & Album Groups):** Provides a high-density, Samsung-inspired UI for grouping physical albums into logical hierarchies.
3.  **Enforcement (Enforcement Engine):** Physically moves or copies files to user-defined destinations, ensuring the disk structure matches the user's intent.

## Strategic Objectives
*   **Physical Sovereignty:** Organizational decisions are reflected in the file system (MOVE/COPY logic).
*   **Inbox Triage:** Dedicated workflow for new media ("Inbox, not Everything Box") based on temporal filters.
*   **High-Density Performance:** A visual experience optimized for 120Hz displays, featuring dense grids and 24dp rounded corners.
*   **Privacy & Speed:** 100% local operation. No cloud dependencies, no AI overhead. Just raw, fast metadata and I/O.

## The Outcome
Users regain control over their storage. Media is not just "viewed"—it is curated into a durable, structured file system that persists independently of the application.
