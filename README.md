# cGallery

cGallery is a simple local gallery app for organizing and browsing image collections and folders.

It’s still in development, but it already works for daily use and I’m improving it version by version.

---

## Current Version

v0.7

---

## What it does right now

- lets you browse images and folders quickly
- organizes things into albums/groups
- handles basic folder structure automatically
- stays fast even with bigger collections
- runs locally (no cloud or anything like that)

## Notes

This isn’t meant to be a full media manager or anything complicated.

It’s just meant to:
- be fast
- stay simple
- not get in the way
- actually make messy folders usable
